# 🍾 GUIA RÁPIDO - RASTREAMENTO POR GARRAFA

## Setup (primeira vez - 10 segundos)

```
Você: "tamanho 500"
Bot:  ✅ Garrafa configurada para 500ml!

Pronto! ✅
```

---

## Usando Todo Dia

### Terminou a garrafa?

```
Você: "Garrafa cheia"
Bot:  🍾 Excelente! Garrafa 2 completa! 
      Total: 1000ml / 3000ml (33%)
      Faltam: 2000ml para a meta!
```

### Bebeu só parte?

```
Você: "Bebida 50%"
Bot:  💧 Registrado! Você bebeu 50% da garrafa
      Total: 1250ml / 3000ml (42%)
```

### Ver progresso

```
Você: "Status"
Bot:  🍾 | 3 garrafas completas (1500ml)
      🍾🍾🍾
      Meta: 1500ml / 3000ml (50%)
```

---

## Mudou de Garrafa?

```
Você: "Tamanho 750"
Bot:  ✅ Tamanho atualizado! 500ml → 750ml
```
